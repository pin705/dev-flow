# 2026-04 pnpm and Docker Dev Flow

This update standardizes local development around `pnpm` and adds a Docker Compose workflow for the web app.

- `pnpm` is now the workspace package manager
- `pnpm-workspace.yaml` is the source of truth for workspace discovery
- `compose.yml` starts the web app and Postgres together
- file polling is enabled for reliable hot reload in Docker
- the web production Dockerfile now understands the monorepo layout
