# CLI Reference

## Auth

```bash
devflow auth login
devflow auth logout
```

`devflow auth login` starts a browser-based device flow. After signing in, approve the device page
so the CLI can fetch bootstrap, read synced history, and upload review artifacts with an approved
session.

## Provider configuration

```bash
devflow config set-provider qwen
```

## Review workflows

```bash
devflow review
devflow review --staged
devflow review --base origin/main
devflow review --files src/a.ts src/b.ts
```

## Explain and tests

```bash
devflow explain src/lib/review.ts
devflow tests src/lib/review.ts
```

## History and diagnostics

```bash
devflow history
devflow doctor
```

## Output modes

Use JSON for scripting and Markdown for artifacts:

```bash
devflow review --base origin/main --json
devflow review --base origin/main --markdown
```
