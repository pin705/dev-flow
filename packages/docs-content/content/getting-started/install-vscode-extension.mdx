# Install the VS Code Extension

The extension is a companion to the CLI. It shells out to the local `devflow` binary and renders results in editor-native panels.

## Setup flow

1. Install the CLI first.
2. Install the VS Code extension.
3. Click **Sign In** in the extension.
4. Complete browser auth and return to the editor.
5. Run **Review Current Changes**.

## Expected commands

The extension uses the same flows as the CLI:

- `devflow review`
- `devflow review --staged`
- `devflow review --files ...`
- `devflow explain <file>`
- `devflow tests <file>`

Read the [VS Code guide](/docs/vscode/guide) for command mapping and result panels.
