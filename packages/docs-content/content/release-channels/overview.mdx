# Release Channels

Devflow should publish coordinated release metadata for both the CLI and VS Code extension.

## Channels

- `stable` for the default production audience
- `preview` for early adopters
- `canary` for internal testing and fast feedback

Each release manifest should include version, checksum, download target, and changelog URL.
